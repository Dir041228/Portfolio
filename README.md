# My Portfolio Website

A personal portfolio website built with Python and Django.

## Live Site
https://adrianpaylande.pythonanywhere.com

## Features
- Home page with personal introduction
- About section
- Skills section
- Projects section
- Contact section

## Setup Instructions
1. Clone the repository:
   git clone https://github.com/Dir041228/Portfolio.git

2. Go into the project folder:
   cd Portfolio

3. Install dependencies:
   pip install -r requirements.txt

4. Run migrations:
   python manage.py migrate

5. Run the development server:
   python manage.py runserver

6. Open browser and go to:
   http://127.0.0.1:8000
